# 💖 Pink Love Proposal Web App

A beautiful, interactive romantic web experience built with Next.js 💕
This project is designed to surprise someone special with a personalized love message, floating hearts, and a fun proposal interaction.

---

## 🌸 Features

* 💗 **Pink Theme UI** – Soft, aesthetic romantic design
* 💖 **Floating Hearts Animation** – Hearts rise with smooth motion
* 🖱️ **Interactive Effects** – Mouse movement affects hearts & background
* ✨ **Water Ripple Effect** – Cursor creates a liquid wave feel
* 💌 **Personalized Name Input** – Enter name on first screen
* 💬 **Dynamic Compliments** – Auto-changing romantic messages
* 😂 **Funny NO Button** – Runs away when hovered
* 💘 **YES Button Celebration** – Shows love message + effects

---

## 🧠 How It Works

1. User enters their name on the front page 💖
2. The app personalizes all messages using that name
3. Romantic compliments are displayed with animations
4. Final question:
   👉 *"Do you love me, [name]?"*
5. The "NO" button escapes 😄
6. Only option left... **YES ❤️**

---

## 🛠️ Tech Stack

* ⚛️ Next.js (App Router)
* 🎨 Tailwind CSS
* 🎥 Framer Motion (for animations)
* 💡 JavaScript (DOM & interactions)

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:9002
```
